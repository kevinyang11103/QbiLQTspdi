## 前言

随着互联网技术的飞速发展，民宿在线预定平台已经成为了旅游行业的重要趋势。本项目旨在为计算机相关专业的学生提供一个实战项目，以Spring Boot和MySQL为主要技术，实现一个功能完善、用户友好的民宿在线预定平台。以下是该项目的详细介绍。

## 内容介绍

本项目是一个基于Java和Spring Boot的民宿在线预定平台，主要功能包括用户注册登录、民宿搜索预订、民宿管理、订单管理等功能。该平台不仅为用户提供了方便快捷的在线预定体验，同时也为民宿主人提供了一个管理和发布民宿信息的平台。通过本项目的开发，学生可以学习到Spring Boot框架的使用、MySQL数据库的构建和管理、前端页面设计与交互等知识。

## 技术介绍

- 语言：Java
- 使用框架：Spring Boot
- 前端技术：JS、Vue、css3
- 开发工具：IDEA/Eclipse
- 数据库：MySQL 5.7/8.0
- 数据库管理工具：phpstudy/Navicat
- JDK版本：jdk1.8
- Maven: apache-maven 3.8.1-bin
- 前端环境：Node.Js 12/14/16

    public boolean addUser(User user) {
        return userDAO.addUser(user);

    public boolean updateUser(User user) {
